# вҡҷпёҸ Installation

### Clone

```bash
git clone https://github.com/Abhishek09821/Youtube-Video-Downloader.git
```

### Backend

```bash
pip install -r requirements.txt

python server.py
```

### Frontend

```bash
cd frontend

npm install

cp .env.example .env

npm run dev
```
